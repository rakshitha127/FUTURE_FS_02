# Mini CRM Lead Management System

A full-stack Mini CRM application developed as part of the Future Interns Internship Program. This project helps businesses manage leads efficiently through a centralized dashboard with authentication, lead tracking, status management, and analytics.

## Features

* Secure admin authentication using JWT
* Lead management dashboard
* Add, edit, and delete leads
* Lead status tracking (New, Contacted, Converted)
* Search and filter functionality
* Follow-up notes management
* Analytics overview cards
* RESTful API integration
* Responsive user interface

## Technologies Used

### Frontend

* React.js
* Vite
* JavaScript
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Authentication

* JSON Web Tokens (JWT)
* bcrypt.js
